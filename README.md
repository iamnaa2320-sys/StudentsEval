# StudentsEval
GPA and Grade Calculator Python Library
Description
GPA&gradeCalculator is a Python utility library that helps students calculate grades, GPA, and CGPA quickly and accurately. The library converts marks or percentages into letter grades and grade points, and computes semester GPA as well as overall CGPA, making academic result calculation simple and reusable across projects.
Technologies Used: Python 
Key Features:Modular Architecture: The project is cleanly structured into separate modules (such as grade_utils.py and cgpa_utils.py) to ensure high code readability, organization, and easy maintenance.

Automated Grade Calculation: Automatically determines the correct letter grades (A, B, C, etc.) and maps them to their respective Grade Points based on the marks entered.

GPA & CGPA Calculator: Seamlessly processes course credit hours and obtained grades across multiple semesters to calculate the exact semester GPA and overall Cumulative GPA (CGPA).

Input Validation & Error Handling: Includes built-in checks to handle invalid data inputs (such as marks out of the standard range) to prevent calculation errors.

Reusable Python Package: Structured as a reusable package, allowing its utility functions to be easily imported and integrated into larger academic or university management systems.
